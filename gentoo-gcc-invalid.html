gcc-config:Active gcc profile is invalid解决办法

Written by Ace King in Gentoo on Tue 02 September 2014. Tags: bugs, gcc,
目录:
  • 错误描述
  • 解决方法：

错误描述

Gentoo软件安装错误,提示：

gcc-config: Active gcc profile is invalid

解决方法：

  • 列出可用的profile
gcc-config -l

gcc-config: Active gcc profile is invalid! [1] i686-pc-linux-gnu-4.6.3

  • 显示当前使用的profile
gcc-config -c

gcc-config: Active gcc profile is invalid! [1] i686-pc-linux-gnu-4.6.3

  • 设置profile
gcc-config i686-pc-linux-gnu-4.6.3

Switching native-compiler to i686-pc-linux-gnu-4.6.3 ... Regenerating /etc/ld.so.cache... [ ok ]
